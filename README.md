# Розробка адаптивного вебсайту для барбершопу "BRITVA" з використанням сучасних технологій фронтенд-розробки.

Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)).
Керівник роботи – **БАРАБАН Дмитро Миколайович**.

## Завдання до дипломної роботи

Продукт має являти собою односторінковий вебзастосунок з високим рівнем продуктивності та унікальним дизайном. Семантична верстка HTML5, стилізація за допомогою CSS3 (або препроцесорів Sass/Less), мова програмування JavaScript (можливе використання бібліотек/фреймворків). Сайт повинен містити інтерактивні елементи (галереї, форми, анімації), бути повністю адаптивним та оптимізованим для швидкого завантаження. За основу взяти структуру та контент сайту http://britva.pl.ua/

## Використані технології

* JavaScript
